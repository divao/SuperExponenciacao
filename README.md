# Super Exponenciacao
Programa desenvolvido como trabalho na disciplina de Projeto e Analise de Algoritmos (2017/2) no curso BCC da UFSCar.

### Desenvolvedor
Caio Ferreira, RA: 726502, Turma A.

## Proposta do projeto
Desenvolver uma animação interativa sobre a resolução de um problema por duas estratégias diferentes de projeto de algoritmos.

## Problema
O problema escolhido foi a exponenciação, utilizando a estratégia força-bruta (que gera um algoritmo O(n)) e o decremento e conquista (que otimiza consideravelmente, gerando um algoritmo O(log n)).
